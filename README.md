# Academic City Fever - Campus Food Ordering System

A modern, responsive food ordering platform designed for campus communities. Built with a clean frontend interface and a robust backend API, this system allows students and staff to browse menus, place orders, and track deliveries seamlessly.

# Features

# For Customers
- **User Registration & Authentication**: Secure login and registration system
- **Interactive Menu**: Browse food items by categories (Breakfast, Lunch, Dinner, Desserts, Drinks)
- **Shopping Cart**: Add, remove, and modify items in cart
- **Order Tracking**: Real-time order status updates with timeline
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices

# For Administrators
- **Admin Dashboard**: Comprehensive order management interface
- **Order Status Updates**: Change order status from pending to completed
- **Analytics Overview**: View order statistics and recent activity
- **User Management**: Monitor user orders and delivery details

# Tech Stack
# Frontend
- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with responsive design
- **Vanilla JavaScript**: Interactive functionality and DOM manipulation

# Backend
- **Node.js**: JavaScript runtime environment
- **Express.js**: Web application framework
- **PostgreSQL**: Relational database for data persistence
- **JWT**: JSON Web Tokens for authentication
- **bcryptjs**: Password hashing and security
- **CORS**: Cross-origin resource sharing

### Deployment
- **GitHub Pages**: Frontend hosting
- **Render**: Backend and database hosting
